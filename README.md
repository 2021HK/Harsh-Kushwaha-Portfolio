## 🔧 Customization

To customize this portfolio for yourself:

1. Update personal information in `index.html`
2. Replace images in `img/` folder
3. Modify colors in `styles.css` (CSS variables)
4. Update projects and experience sections
5. Change Formspree form ID in contact section

## 📧 Contact Form Setup

The contact form uses Formspree. To set up your own:

1. Sign up at [Formspree.io](https://formspree.io)
2. Create a new form
3. Replace the form action in `index.html`:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```
